# Todo App - React 

A clear and simple todo App built with react.
Add, Edit, Mark as Completed, and Delete tasks with data saved in browser LocalStorage.

---

### Todo App Live Link

[https://todolistrapp.netlify.app](https://todolistrapp.netlify.app)

---

## Features

- Add New Tasks
- Edit Existing Tasks
- Mark Tasks as Completed
- Delete Tasks
- Fully Responsive UI

---

## Tech Stack

| Technology | Used For |
|------------|----------|
| React.js | UI & State Management |
| LocalStorage | Data Persistence |
| JavaScript | Logic |
| CSS | Styling |
